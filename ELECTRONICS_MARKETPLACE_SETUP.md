## üíæ Database Schema

### listings table (for electronics)
```sql
CREATE TABLE listings (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL,
    category VARCHAR(100),           -- e.g., "Mobiles & Smartphones"
    subcategory VARCHAR(100),        -- e.g., "iPhone"
    title VARCHAR(255),              -- e.g., "iPhone 13 Pro 256GB"
    description TEXT,
    price INT,
    location VARCHAR(100),
    phone VARCHAR(20),
    email VARCHAR(100),
    condition VARCHAR(50),           -- e.g., "Used", "Like New", "New"
    photos VARCHAR(255),             -- File paths
    created_at TIMESTAMP,
    status ENUM('active', 'sold', 'pending', 'archived'),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```
